# yumemi-frontend-population-chart

## ローカル環境

画面起動
```bash
npm start
```


## テスト（playwrightというライブラリを使用）

テストコマンドを実行

```bash
npx playwright test
```

テスト結果をブラウザに表示

```bash
npx playwright show-report 
```
